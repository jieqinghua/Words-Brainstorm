# 创意方法论库

每次脑暴从以下 5 种方法中随机选取一种。每种方法提供了核心逻辑、发散策略和示例。

---

## 方法一：随机联想

**核心逻辑**：从一个完全无关的随机词出发，强制寻找与主题词的连接点。利用大脑的"模式识别"本能，在看似无关的事物间建立桥梁。

**发散策略**：
1. 在心里生成一个远离主题词语义场的随机词
2. 列出两个词各自的关键属性（形状、颜色、功能、场景、文化含义）
3. 寻找属性列表中的交叉项
4. 基于交叉项构建组合

**示例**（含推导路径）：
- 主题词"水" + 随机词"刀" → 推导：水的属性"高压、流动" + 刀的属性"切割、锋利" → **"水刀"**（真实存在的工业切割技术，水的力量 + 刀的切割）
- 主题词"书" + 随机词"鸟" → 推导：书"打开如展开、纸页" → 鸟"翅膀、飞翔" → **"展卷"**（展开书卷如鸟展翅，已有词汇，关联自然）
- 主题词"电" + 随机词"老虎" → 推导：电"驱动、游戏厅" → 老虎"猛兽、图案" → **"老虎机"**（电驱动的赌博游戏机，老虎图案是标志）

**注意事项**：随机词要真正随机，不要下意识选相似词。

---

## 方法二：远距联想

**核心逻辑**：从一个看似很远但存在深层联系的领域选词，找到那个"隐藏的共同点"。不同于随机联想的完全随机，远距联想追求"远但终究能连上"。

**发散策略**：
1. 确定主题词所属的领域（如"电"→ 能源/物理）
2. 跳到一个相隔很远的领域（如"生物""艺术""饮食"）
3. 在该领域中选一个有特征的词
4. 寻找跨领域的结构相似性或功能类比

**示例**（含推导路径）：
- 主题词"代码" → 跳到领域"建筑" → 选词"钢筋" → 推导：代码支撑系统如钢筋支撑建筑 → **"架构"**（程序员和建筑师共用这个词，关联天然成立）
- 主题词"时间" → 跳到领域"烹饪" → 选词"慢炖" → 推导：时间让食材入味如经历让人沉淀 → **"陈酿"**（时间的烹饪，酒类术语借用表达岁月积累）

**注意事项**：领域跳跃越大越好，但要确保有类比空间。

## 方法三：概念组合

**核心逻辑**：将两个词分别拆解为概念要素，然后交叉重组。不是简单的词拼接，而是概念层面的化学反应。

**发散策略**：
1. 拆解主题词的概念要素（是什么、干什么、给谁用、在什么场景）
2. 选一个发散词，同样拆解
3. 将两组的要素进行交叉配对
4. 选出最有化学反应的一组，形成新概念

**示例**（含推导路径）：
- 主题词"咖啡"（提神、液体、苦、早晨）+ 发散词"枕头"（休息、柔软、床、夜晚）
  → 交叉配对："苦+夜晚" → **"午夜咖啡"**（加班场景，苦味与深夜相伴）
  → 交叉配对："液体+柔软" → **"咖啡枕"**（咖啡渣填充的抱枕，有香气，真实产品）
- 主题词"音乐"（旋律、节奏、情感、耳朵）+ 发散词"药"（治疗、剂量、苦、身体）
  → **"音乐处方"**（用旋律治疗情绪，已有概念，自然不牵强）

**注意事项**：要素拆解得越细，组合空间越大。

---

## 方法四：反向思维

**核心逻辑**：从主题词的反义词或对立面出发，经过反转后再建立联系。创意往往诞生于矛盾和张力之中。

**发散策略**：
1. 找出主题词的反义词或对立概念
2. 基于反义词生成一个发散词（不完全等于反义词，而是反义词的近邻）
3. 思考"如果这个对立面和主题词同时存在会怎样"
4. 从矛盾中提炼创意组合

**示例**（含推导路径）：
- 主题词"光" → 反义"暗" → 发散词"影子" → 推导：光与影子永远共存，互相定义 → **"光影"**（天然CP，美术与摄影的核心概念）
- 主题词"快" → 反义"慢" → 发散词"蜗牛" → 推导：快递+蜗牛=反差萌 → **"蜗牛快递"**（慢的蜗牛送快的快递，自带故事感）

**注意事项**：不要直接用反义词做发散词，要绕一步，增加意外感。

---

## 方法五：属性嫁接

**核心逻辑**：提取主题词的一个核心属性，嫁接到一个看似不兼容的对象上，产生"杂交"式的创意。

**发散策略**：
1. 提取主题词的 5 个属性（物理属性、功能属性、情感属性、文化属性）
2. 随机选择其中一个属性
3. 找一个不具备该属性的词，强行把属性嫁接过去
4. 想象嫁接后的效果，提炼创意表达

**示例**（含推导路径）：
- 主题词"火" → 属性"红色" → 嫁接到"珊瑚" → 推导：火的颜色→红→红珊瑚是真实存在的珍贵宝石 → **"红珊瑚"**（火的颜色穿在珊瑚身上，自然成立）
- 主题词"火" → 属性"加热" → 嫁接到"花甲" → 推导：火的功能→烧烤→花甲可以烤→夜宵摊的烟火气 → **"花甲烧烤"**（真实生活场景，火的温度和花甲的鲜味结合）
- 主题词"风" → 属性"无形" → 嫁接到"墙" → 推导：风看不见但能挡住→无形的屏障 → **"风墙"**（气流的屏障，空气动力学中的真实概念）
- 主题词"梦" → 属性"虚幻" → 嫁接到"货币" → 推导：虚拟世界的流通价值→游戏币、元宇宙 → **"梦币"**（虚拟世界的流通物，已有类似概念）

**注意事项**：嫁接后的组合要有想象空间，不能太牵强。

---

## 方法选用建议

- **追求意外感** → 优先用随机联想或反向思维
- **追求内涵深度** → 优先用远距联想或概念组合
- **追求画面感** → 优先用属性嫁接
- 多次使用同一方法时，通过换领域、换属性、换反义方向来保证多样性
